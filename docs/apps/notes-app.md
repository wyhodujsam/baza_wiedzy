# notes-app

> **TL;DR:** Aplikacja do notatek z CRUD (dodawanie, usuwanie, brak edycji). Dane w pliku JSON, bez bazy danych.

- **Katalog:** ~/notes-app
- **Stack:** Java 17, Spring Boot 3.2.5, Thymeleaf
- **Port:** 8081
- **Uruchomienie:** `cd ~/notes-app && ./mvnw spring-boot:run` lub `mvn spring-boot:run`

Dane przechowywane w pliku `notes.json`.
